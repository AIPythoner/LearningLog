## git常用命令
1.**首次使用GitHub建立仓库**

    git init 初始化仓库 
    git add README.md 保存需要提交的文件
    git commit -m "first commit" 提交的备注
    git remote add origin https://github.com/AIPythoner/test.git
    git push -u origin master

2.**日常提交**

    git add .
    git commit -m "first commit"
    git push -u origin master
    
3.**常用命令**

    git clone https://demo.git
